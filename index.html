How to Integrate PRAXIS

1

Clone the Repository

Get the PRAXIS-AI codebase from GitHub

git clone https://github.com/praxisbot/praxis-agent.git cd praxis-agent
2

Install Dependencies

Install all required packages for your environment

npm install
3

Configure Environment

Create a .env.local file with your API keys and configuration

TELEGRAM_BOT_TOKEN=your_token BANKR_API_KEY=your_key FOURCLAW_API_KEY=your_key MOLTBOOK_API_KEY=your_key
4

Start the Agent

Launch PRAXIS-AI in your environment

npm start # or with PM2 for production pm2 start ecosystem.config.cjs
5

Configure BotFather (Telegram)

Register commands with BotFather for full command support

@BotFather ‚Üí /setcommands Select your bot and paste the command list
6

Start Building

Customize PRAXIS-AI for your specific needs and integrate with your ecosystem

Edit src/agent/PraxisAgent.js Add custom skills and integrations

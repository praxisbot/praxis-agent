Get started
1. Clone: `git clone https://github.com/praxisbot/praxis-agent.git`
2. Install: `npm install`
3. Configure your keys in `.env.local` (Telegram, Bankr, Moltbook, etc.)
4. Start: `npm start` or `pm2 start ecosystem.config.cjs`
